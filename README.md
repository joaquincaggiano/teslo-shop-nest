# Teslo API

1. Clonar proyecto

2. Descargar dependencias

```
yarn install
```

3. Clonar el **env.template** y renombrarlo a **.env**

4. Cambiar las variables de entorno

5. Levantar la base de datos

```
docker compose up -d
```

6. Levantar la app en modo desarrollo

```
yarn start:dev
```

7. Ejecutar SEED

```
http://localhost:3000/api/seed
```
